Implementation of Dijkstra Algorithm

Problem description:

You are required to implement Dijikstra algorithm to find the length of a shortest path from a given source vertex to each of the remaining vertices in an undirected graph, using a simple data structure, say simple scheme, as well as Fibonacci heap, say f-heap scheme, and measure the relative performance of the two implementations. Both of the schemes MUST use the adjacency lists representation for graphs. The simple scheme uses an array to determine the next edge to include while the f‐heap scheme uses a Fibonacci heap to do this. The simple scheme should run in O (n^2) time, where n is the number of vertices in the graph.

C++ Code

Structure of program:

1. Random input mode (function “randmode”):

  • 1. Type in the number of vertex: n, edge density: d and source node: x.
  • 2. Generate graph.
  • 3. Simple scheme to implement dijkstra algorithm.
    1. Define a array called “dist[]” with initial value infinity.
    2. dist[i] equals to the weight of node i in the graph.
    3. For each node in the graph except the source node: Find the shortest path from source to other nodes. If the cost of path through one node’s adjacent node is less than the cost of previous path, then update the dist value of that node.
  • 4. Fibonacci heap scheme to implement dijkstra algorithm.
    1. Define a array called “dist[]” with initial value infinity.
    2. Initialize a new Fibonacci heap with function “heap_initial”.
    3. For each node in graph: Function “cre_nd” create new Fibonacci node with label i and weight dist[i], and insert it into the heap using function “Insert”.
    4. Make dist value of source node equals to 0. Function “de_key” is for the key of source node to dist[source], which is 0.
    5. Function “Findmin” finds the node u with minimum weight. “ex_min” extracts that node and the update the minimum pointer of the heap.
    6. For each nerghbor v of u: if dist[v] is larger than dist[u] plus weight between u and v, then update dist[v] and “de_key” function updates weight of node v in the heap.

2. User input mode(function “fileinput")

  1. Read the file with ifstream.
  2. Use “getline” to store each line in the file into a string.
  3. First line is the source node x, second line is number of vertex and edge, and the rest are the edges needed to be added into the graph. “addedge” function adds all the edges into the graph.
  4. Simple scheme to implement dijkstra algorithm.
  5. Fibonacci heap scheme to implement dijkstra algorithm.
